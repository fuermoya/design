# element-plus-vite-starter

> 基于 Vite 的 Element Plus 启动模板

- 预览地址: <https://vite-starter.element-plus.org>

这是一个使用 [unplugin-vue-components](https://github.com/antfu/unplugin-vue-components) 实现按需导入 element-plus 的示例。

> 如果您想要全量导入，可能会非常简单，不需要示例。只需按照 [快速开始 | 文档](https://element-plus.org/zh-CN/guide/quickstart.html) 进行导入即可。

如果您只想要一个手动按需导入的示例，可以查看 [unplugin-element-plus/examples/vite](https://github.com/element-plus/unplugin-element-plus/tree/main/examples/vite)。

如果您想要一个 nuxt 启动模板，请查看 [element-plus-nuxt-starter](https://github.com/element-plus/element-plus-nuxt-starter/)。

## 项目设置

```bash
pnpm install

# npm install
# yarn install
```

### 开发环境编译和热重载

```bash
npm run dev
```

### 生产环境编译和压缩

```bash
npm run build
```

## 使用方法

```bash
git clone https://github.com/element-plus/element-plus-vite-starter
cd element-plus-vite-starter
npm i
npm run dev
```

### 自定义主题

请查看 `src/styles/element/index.scss`。
